# ArchiTECH — BIM × Tech × AI × Architecture

Portfolio of **Navneet Singh** — architect and builder of ArchiTECH, an agentic
BIM operating system: Revit automation, local AI agents, and knowledge systems
working as one governed command layer.

**Live site:** https://architech97.github.io/architech-portfolio/

## Highlights

- True-3D wireframe scenes on Canvas 2D — no frameworks, no dependencies
- A Burj Khalifa-inspired form study you can orbit, and a parametric twist
  tower that assembles itself in a looping cinematic sequence
- Real, measured numbers from the shipped ArchiTECH ecosystem — 66 Revit ribbon
  tools, an 88-tool Revit MCP connector, a 6,991-note knowledge brain, a
  bilingual voice agent
- Opt-in synthesized ambience and interaction sound (Web Audio, no audio files)
- Reduced-motion support, keyboard accessible, fully static — open
  `index.html` in any modern browser
